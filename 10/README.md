"Автоматическая сортировка файлов: Напишите скрипт, который автоматически сортирует файлы в указанной директории:
Файлы с расширениями .jpg, .png и .gif перемещает в папку Images.
Документы (.txt, .pdf, .docx) — в папку Documents.
Скрипт должен запускаться автоматически через cron каждую ночь и вести лог всех перемещений."


Чтобы скрипт запускался автоматически, нужно настроить cron

1 chmod +x script.sh

2 crontab -e

3.Добавить  "0 0 * * * script.sh test"
(время путь_к_скрипту путь_к_папке)
